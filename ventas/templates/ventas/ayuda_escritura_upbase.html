Prepare Up Base

Coloque el cursor en la primera celda de código. No toque el teclado hasta que termine.

Detención de emergencia: mueva el mouse a una esquina de la pantalla para activar el seguro de PyAutoGUI.

Qué hará por cada fila

  1. Escribe el código.
  2. Presiona Enter para validar el producto.
  3. Sube una fila y avanza dos columnas hacia cantidad.
  4. Escribe la cantidad.
  5. Presiona Enter y continúa con el siguiente código.

La escritura usa pegado rápido por portapapeles. Si Up Base no alcanza a validar, cambie de Turbo/Rápida a Normal o Segura.
